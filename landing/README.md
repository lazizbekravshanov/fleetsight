# FleetSight Landing (Next.js)

Redesigned premium landing page for FleetSight using:

- Next.js (App Router)
- Tailwind CSS
- Framer Motion
- Responsive layout
- Dark mode toggle
- Semantic HTML + accessible controls
- SEO metadata

## Run

```bash
cd landing
npm install
npm run dev
```

Open:

`http://localhost:3000`

## Build

```bash
npm run build
npm run start
```
