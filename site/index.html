FleetSight

Detect likely affiliated and chameleon carrier networks with transparent, explainable scoring.

Try Sample Data View GitHub
/fleetsight analyze carriers.csv --top 50 --threshold 30

How It Works

01

Ingest CSV

Read carrier records with graceful handling for blank fields and strict required-column checks.

02

Score Links

Normalize phone, email, domain, address, and IP. Weighted matching with rarity down-weighting.

03

Build Clusters

Threshold pairwise edges, compute connected components, export JSON and CSV reports.

What You Get

Ranked Links

Top N carrier-to-carrier links with total score and contribution-level reasons.

links.json / links.csv

Affiliation Clusters

Connected components over score-thresholded edges, including size and link stats.

clusters.json / clusters.csv

Chat Summary

Concise markdown result with top links, top clusters, and report paths.

summary.md
